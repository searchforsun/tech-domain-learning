# ROADMAP.md 与 PROGRESS.md 模板

## ROADMAP.md

```markdown
# <技术领域> 学习路线

> 生成日期：YYYY-MM-DD · 修订：YYYY-MM-DD
> 路线基于公开资料整理，以官方文档为准。

## 前置知识

| 领域 | 建议具备 |
|------|-----------|
| … | … |

## 本路线定位

| 取向 | 说明 |
|------|------|
| 弱化 | 本路线不深入的方向（避免读者期待偏差） |
| 强化 | 本路线重点关注的方向 |
| 核心抓手 | 学完能做什么、能看懂什么 |

## 路线图

```
阶段 1：<主题>
        ↓
阶段 2：<主题>
        ↓
阶段 3：<主题>
        ↓
阶段 4+：<主题>（按需延伸至 8 阶段；窄专题可少至 2–3 阶段）
```

## 阶段总览

| 阶段 | 名称 | 核心主题 | 建议周期 |
|------|------|----------|----------|
| 1 | … | … | X–Y 周 |
| 2 | … | … | X–Y 周 |

## 各阶段详情

### 阶段 N：<阶段名>

- **主题**：本阶段覆盖的核心知识点
- **反模式**：常见的理解偏差或错误用法
- **实践**：建议动手做什么
- **建议周期**：X–Y 周（取决于深度和现有基础）
- **阅读**：官方文档入口（WebSearch 确认）
```

## PROGRESS.md

```markdown
# <技术领域> 学习进度

- 技术方向：…
- 路线文件：[ROADMAP.md](ROADMAP.md)
- 当前阶段：阶段 N — <阶段名>（未开始 / 进行中 / 已完成）
- 上次更新：YYYY-MM-DD

## 阶段清单

| 阶段 | 名称 | 状态 | 完成日期 | 时长 |
|------|------|------|----------|------|
| 1 | … | 未开始 | — | — |
| 2 | … | 未开始 | — | — |

## 完成记录

（每完成一个阶段追加一条：日期、时长、2–5 条收获）

## 下一阶段入口

- 理论：[`0N-阶段名/THEORY.md`](0N-阶段名/THEORY.md)
- 示例：[`0N-阶段名/demo/README.md`](0N-阶段名/demo/README.md)
```
